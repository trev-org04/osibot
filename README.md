# OSIBOT V1

A Large Language Model (LLM) trained on [OpenWebText](https://huggingface.co/datasets/Skylion007/openwebtext) to diagnose problems with networking infrastructure.


## Tech Stack

**Language:** `Python`

**Dependencies:** `PyTorch v2.2.2, Numpy v1.26.4`
